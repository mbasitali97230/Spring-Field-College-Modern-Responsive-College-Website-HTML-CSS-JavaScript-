# 🎓 Spring Field College Website

A modern, fully responsive multi-page college website built using pure HTML, CSS, and JavaScript.

This project demonstrates advanced frontend design concepts including scroll animations, glassmorphism UI, SPA-style page switching, and a fully functional mobile dropdown navigation system.

## ✨ Features

- Fully Responsive Design
- Scroll-triggered animations (Intersection Observer)
- Glassmorphism UI effects
- Sticky animated navbar
- Complete mobile dropdown navigation system
- Multi-page SPA structure
- Campus slider section
- Modern hero section with statistics
- Clean footer layout
- SEO-friendly semantic structure
- Built without any frameworks

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid, Glass Effects)
- Vanilla JavaScript (DOM, Events, Intersection Observer)

## 📱 Mobile Features

- Animated hamburger menu
- Smooth dropdown navigation
- Auto close on link click
